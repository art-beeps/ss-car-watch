Elektroauto uzlādes stacija 11kW, 3 fāzes, 16A, Type 2, kabelis 5 m
Sertificētu elektroauto uzlādes staciju ar Type 2 kabeli. Piemērota drošai un ērtai elektroauto uzlādei mājās, uzņēmuma teritorijā vai privātā stāvvietā.
Galvenās īpašības:
:: Uzlādes jauda: 11 kW
:: Pieslēgums: trīsfāžu, 16A, Mode 3
:: Savienojums: Type 2 kabelis, 5 m, Eiropas standarts
:: Barošana: 85–415 V AC, 50/60 Hz, 16A 3P+N+PE
:: Vadība: Bluetooth 4.2 + mobilā lietotne KW22
:: Lietotnē iespējams palaist un apturēt uzlādi, plānot uzlādes laiku, sekot patēriņam un saņemt paziņojumus
:: RFID karšu lasītājs piekļuves kontrolei
:: Aizsardzība: IP55, piemērota uzstādīšanai ārā
:: Ugunsdrošības klase: UL94V-0
:: Darba temperatūra: no -25 līdz +50 C
Priekšrocības:
:: Sertificēta un droša uzlādes stacija ikdienas lietošanai
:: Piekļuves kontrole ar RFID kartēm
:: Uzlādi var atļaut tikai konkrētiem lietotājiem
:: Papildu RFID kartes citiem lietotājiem iespējams iegādāties atsevišķi
:: Ērta vadība ar bezmaksas mobilo lietotni
:: Piemērota gan privātmājām, gan uzņēmumiem
:: Izturīgs IP55 korpuss lietošanai ārā

Станция зарядки для электромобилей 11kW, 3 фазы, 16A, Type 2, кабель 5 м
Продаётся новая сертифицированная станция зарядки для электромобилей с кабелем Type 2. Подходит для безопасной и удобной зарядки электромобиля дома, на территории предприятия или на частной парковке.
Основные характеристики:
:: Мощность зарядки: 11 кВт
:: Подключение: трёхфазное, 16A, Mode 3
:: Разъём: кабель Type 2, 5 м, европейский стандарт
:: Питание: 85–415 V AC, 50/60 Hz, 16A 3P+N+PE
:: Управление: Bluetooth 4.2 + мобильное приложение KW22
:: В приложении можно запускать и останавливать зарядку, планировать время зарядки, контролировать потребление и получать уведомления
:: Считыватель RFID-карт для контроля доступа
:: Защита: IP55, подходит для установки на улице
:: Класс огнестойкости: UL94V-0
:: Рабочая температура: от -25 до +50 C
Преимущества:
:: Сертифицированная и безопасная зарядная станция для ежедневного использования
:: Контроль доступа с помощью RFID-карт
:: Зарядку можно разрешить только конкретным пользователям
:: Дополнительные RFID-карты для других пользователей можно приобрести отдельно
:: Удобное управление через бесплатное мобильное приложение
:: Подходит для частных домов и предприятий
:: Прочный корпус IP55 для использования на улице

Настенное крепление и болты
