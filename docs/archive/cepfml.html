Elektroauto uzlādes stacija Pni Kw22, 22 kW, Type 2, 5 m kabelis, Bluetooth, Rfid, CE
Pārdodam CE sertificētu elektroauto trīsfāžu uzlādes staciju PNI KW22 ar Type 2 kabeli, Bluetooth vadību un RFID piekļuves kontroli.
Uzlādes stacija piemērota lietošanai mājās, uzņēmumā, autoservisā, autostāvvietā vai citā vietā, kur nepieciešama droša un kontrolējama elektroauto uzlāde.
Galvenie parametri:
:: Modelis: PNI KW22
:: Uzlādes jauda: līdz 22 kW
:: Pieslēgums: 3 fāzes
:: Uzlādes strāva: regulējama 10A / 13A / 16A / 20A / 32A
:: Uzlādes režīms: Mode 3
:: Auto savienojums: Type 2
:: Kabeļa garums: 5 m
:: Barošana: 85–415 V AC, 50/60 Hz
:: Vadība: 3.5 collu skārienekrāns
:: Savienojums ar telefonu: Bluetooth 4.2
:: Mobilā lietotne: KW22
:: RFID karšu lasītājs: uzlādes aktivizēšanai ar RFID karti
:: Aizsardzība: IP55
:: Korpusa ugunsdrošība: UL94V-0
:: Darba temperatūra: -25 līdz +50 C
:: CE sertifikācija: jā
Strāvas regulēšana, lai nepārslogotu elektrotīklu:
Uzlādes stacijai iespējams samazināt uzlādes strāvu. Ja elektroinstalācija vai pieslēguma jauda nav pietiekama 32A režīmam, uzlādi var iestatīt, piemēram, uz 16A, 13A vai 10A. Tas palīdz samazināt slodzi uz mājas vai uzņēmuma elektrotīklu un ļauj drošāk izmantot uzlādi vietās ar ierobežotu pieslēguma jaudu.
Strāvas stiprumu iespējams mainīt:
:: manuāli uz pašas uzlādes stacijas ekrāna
:: caur Bluetooth mobilo lietotni KW22
Bluetooth lietotnes iespējas:
:: uzlādes palaišana un apturēšana
:: uzlādes laika plānošana
:: uzlādes strāvas iestatīšana
:: patēriņa uzskaite
:: uzlādes statusa kontrole
:: paziņojumi telefonā
RFID piekļuves kontrole:
Uzlādi iespējams aktivizēt ar RFID karti. Tas ir ērti, ja uzlādes staciju izmanto vairāki lietotāji vai ja nevēlaties, lai staciju varētu palaist jebkurš garāmgājējs.
RFID sistēma ļauj kontrolēt:
:: kurš lietotājs veicis uzlādi
:: kad uzlāde sākta
:: cik ilgi tā veikta
:: cik elektroenerģijas patērēts
Komplektācijā ietilpst:
:: Uzlādes stacija PNI KW22
:: 5 metri garš Type 2 kabelis
:: Sienas stiprinājuma plāksne
:: Skrūves
:: Lietotāja rokasgrāmata
:: RFID karte
Papildu RFID piekļuves kartes iespējams iegādāties atsevišķi.

Зарядная станция для электромобиля PNI KW22, 22 кВт, Type 2, кабель 5 м, Bluetooth, RFID, CE
Продаём CE-сертифицированную трёхфазную зарядную станцию для электромобиля PNI KW22 с кабелем Type 2, управлением через Bluetooth и RFID-контролем доступа.
Зарядная станция подходит для использования дома, на предприятии, в автосервисе, на парковке или в другом месте, где требуется безопасная и контролируемая зарядка электромобиля.
Основные параметры:
:: Модель: PNI KW22
:: Мощность зарядки: до 22 кВт
:: Подключение: 3 фазы
:: Зарядный ток: регулируемый 10A / 13A / 16A / 20A / 32A
:: Режим зарядки: Mode 3
:: Подключение к автомобилю: Type 2
:: Кабель: 5 м
:: Питание: 85–415 V AC, 50/60 Hz
:: Управление: сенсорный экран 3.5"
:: Подключение к телефону: Bluetooth 4.2
:: Мобильное приложение: KW22
:: Считыватель RFID-карт: для активации зарядки с помощью RFID-карты
:: Степень защиты: IP55
:: Огнестойкость корпуса: UL94V-0
:: Рабочая температура: от -25 до +50 C
:: CE-сертификация: да
Регулировка тока, чтобы не перегружать электросеть:
В зарядной станции можно уменьшить зарядный ток. Если электропроводка или доступная мощность подключения не рассчитаны на режим 32A, зарядку можно установить, например, на 16A, 13A или 10A. Это помогает снизить нагрузку на домашнюю или корпоративную электросеть и безопаснее использовать зарядку в местах с ограниченной мощностью подключения.
Зарядный ток можно изменить:
:: вручную на экране зарядной станции
:: через мобильное приложение KW22 по Bluetooth
Возможности Bluetooth-приложения:
:: запуск и остановка зарядки
:: планирование времени зарядки
:: настройка зарядного тока
:: учёт потребления электроэнергии
:: контроль статуса зарядки
:: уведомления на телефон
RFID-контроль доступа:
Зарядку можно активировать с помощью RFID-карты. Это удобно, если зарядной станцией пользуются несколько человек или если нужно ограничить доступ посторонних пользователей.
RFID-система позволяет контролировать:
:: какой пользователь выполнял зарядку
:: когда началась зарядка
:: сколько времени она длилась
:: сколько электроэнергии было использовано
В комплект входит:
:: Зарядная станция PNI KW22
:: Кабель Type 2, 5м
:: Настенная монтажная пластина
:: Крепёжные винты
:: Руководство пользователя
:: RFID-карта
Дополнительные RFID-карты доступа можно приобрести отдельно.
